qemu运行riscv linux

过程：
在物理服务器上安装qemu模拟器，模拟器中运行基于riscv指令集编译的linux镜像文件。

工具集合：

  • riscv-qemu（模拟器，可以模拟运行riscv指令集的程序或镜像）
  • riscv-tools（基于riscv指令集的交叉编译工具）
  • riscv-pk（用于包装内核文件vmlinux）
  • busybox(用于给linux镜像安装基本命令，如ls，cat，mv等等)
